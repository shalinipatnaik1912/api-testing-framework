# api-testing-framework

REST API test automation framework using Python, Pytest, and the Requests library.

## Description

This repository contains a lightweight API testing framework that organizes test cases, configuration, and report output to simplify REST API validation. It is designed to help automate API tests using standard Python tools, including pytest for test execution and assertions.

## Repository Structure

- `config/` - configuration modules and shared settings
- `reports/` - generated test results and HTML reports
- `schemas/` - JSON schema definitions for API response validation
- `tests/` - pytest test cases for API endpoints
- `utils/` - helper utilities and reusable code
